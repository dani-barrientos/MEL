Detailed Description

Executor specialization using NaiveInlineExecutionAgent as execution agent.

Its behaviour is same as InlineExecutor, but made in the most easy way, while InlineExecutor has better performance (also it shouldn't be very importan) because it avoids unnecesary copies and moves by reimplementing base Executor functions
